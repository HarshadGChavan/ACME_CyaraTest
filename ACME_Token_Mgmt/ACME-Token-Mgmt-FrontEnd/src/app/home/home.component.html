Log in using admin user (admin/admin@123) and View all tokens with more details and enable/disable them.

  • Backend created with Static Database of List, which can store Tokens created dynamically.
  • FrontEnd created with Admin user who can login and manage tokens.
  • FrontEnd can be used by any user to validate his token.

Technical Stack Used :

Front End:

  • Angular 11
  • BootStrap
  • Angular Material Table (for paging and filter)
  • Toaster Service

Back End:

  • .Net 5
  • AutoMapper
  • Repository Pattern

Note: We can still refine the code and implement proper validation with reactive forms on frontend. Also on backend can strength the error handling by custom error handler middleware. Unit test case can be implement as well.
